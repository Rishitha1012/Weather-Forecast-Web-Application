A simple weather application that fetches real-time weather data using the OpenWeatherMap API.

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## Features
- Search weather by city name
- Displays temperature, weather condition, humidity, and wind speed
- Clean and simple UI

## How It Works
1. Enter a city name
2. Click on "Check Weather"
3. View real-time weather details

## API Note
- This project uses OpenWeatherMap API
- API key activation may take some time after creation
